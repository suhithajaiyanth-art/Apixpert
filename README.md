
# Apixpert

This project is a modern, production-ready React (Vite) application for course listings, job portal, and AI-powered CADGPT features.

## Features
- Modern, compact homepage
- Most Popular Courses section
- Job Portal with professional, animated layout
- CADGPT: AI prompt interface for CAD automation
- Responsive design with Tailwind CSS
- Vercel deployment ready

## Getting Started
1. Install dependencies:
	```
	npm install
	```
2. Start the development server:
	```
	npm run dev
	```
3. Build for production:
	```
	npm run build
	```

## Deployment
This project is configured for seamless deployment on Vercel. Push to the main branch to trigger redeployment.
